Minigame Mayhem Playtesting Guide (Android)

For some reason Google took down the official playtesting group, so until I can get it reinstated, you can directly download the current version here.

In order to gain access to the beta testing build through the Play Store, Google requires me to have testers added to a Google Group, the steps below will show you how you can join it and start testing!

  1. Go to the group page
  2. Sign in with the same Google account you use in the Google Play Store
  3. Press the Join group button next to the group name
  4. Press the Join group button in the pop-up that appears

You should now have access to the Minigame Mayhem Early Access page! If the link is still blocked for you, wait a few minutes for the changes to update on Google’s end, and make sure you’re using the right account.
